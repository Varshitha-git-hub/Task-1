Python Programming Concepts

Python Intro

Definition: Python is a high-level, interpreted, and general-purpose programming language.

print("Hello, Python!")

Python Get Started

Definition: To run Python programs, install Python and use an IDE or text editor like VS Code or PyCharm.

Python Syntax

Definition: Python syntax is simple and easy to understand. Code blocks are defined by indentation.

x = 10
if x > 5:
    print("Greater than 5")

Python Comments

Definition: Comments help explain code and are ignored during execution.

# This is a comment
print("Hello, world!")

Python Variables

Definition: Variables store data.

name = "John"
age = 25
print(name, age)

Python Data Types

Definition: Python supports various data types like int, float, string, list, tuple, etc.

x = 10  # int
y = 10.5  # float
name = "Alice"  # string

Python Numbers

Definition: Python supports numbers including integers, floats, and complex numbers.

a = 5       # int
b = 5.7     # float
c = 3 + 4j  # complex

Python Casting

Definition: Casting is the process of converting one data type to another.

a = "10"
b = int(a)  # Convert string to integer
print(b)

Python Strings

Definition: Strings are sequences of characters enclosed in single or double quotes.

greeting = "Hello"
print(greeting)

Python Booleans

Definition: Booleans represent either True or False values.

is_active = True
is_logged_in = False

Python Operators

Definition: Operators are used to perform operations on variables and values.

a = 10
b = 5
print(a + b)  # Addition

Python Lists

Definition: Lists are mutable collections of ordered items.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Access first item

Python Tuples

Definition: Tuples are immutable collections of ordered items.

coordinates = (10, 20)
print(coordinates[1])  # Access second item

Python Sets

Definition: Sets are unordered collections of unique items.

colors = {"red", "green", "blue"}
print(colors)

Python Dictionaries

Definition: Dictionaries store key-value pairs.

person = {"name": "John", "age": 30}
print(person["name"])

Python If...Else

Definition: Conditional statements used for decision making.

x = 10
if x > 5:
    print("Greater than 5")
else:
    print("Less than or equal to 5")

Python Match

Definition: Pattern matching introduced in Python 3.10.

match x:
    case 1:
        print("One")
    case _:
        print("Other")

Python While Loops

Definition: A `while` loop runs as long as the condition is true.

x = 0
while x < 5:
    print(x)
    x += 1

Python For Loops

Definition: A `for` loop iterates over a sequence.

for fruit in ["apple", "banana", "cherry"]:
    print(fruit)

Python Functions

Definition: Functions are reusable blocks of code.

def greet(name):
    print("Hello " + name)

greet("Alice")

Python Lambda

Definition: A lambda function is an anonymous function.

square = lambda x: x * x
print(square(5))

Python Arrays

Definition: Arrays are more efficient for numerical data than lists.

from array import array
arr = array('i', [1, 2, 3])

Python Classes/Objects

Definition: Classes are blueprints for creating objects.

class Dog:
    def __init__(self, name):
        self.name = name

dog = Dog("Buddy")
print(dog.name)

Python Inheritance

Definition: Inheritance allows one class to inherit from another.

class Animal:
    def sound(self):
        print("Animal sound")

class Dog(Animal):
    def sound(self):
        print("Bark")

d = Dog()
d.sound()

Python Iterators

Definition: Iterators are objects that can be iterated over.

numbers = [1, 2, 3]
iter_numbers = iter(numbers)
print(next(iter_numbers))

Python Polymorphism

Definition: Polymorphism allows different classes to share a common interface.

class Cat:
    def sound(self):
        return "Meow"

class Dog:
    def sound(self):
        return "Bark"

def make_sound(animal):
    print(animal.sound())

make_sound(Cat())  # Meow
make_sound(Dog())  # Bark

Python Scope

Definition: Scope refers to the visibility of variables.

def my_function():
    x = 10  # Local scope
    print(x)

my_function()

Python Modules

Definition: Modules are Python files that contain functions, variables, and classes.

import math
print(math.sqrt(16))
